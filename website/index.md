# WeatherPlanet Fan Creators (WPFC)

WeatherPlanet Fan Creators（以下「WPFC」）は、WeatherPlanetのファンによる非公式な創作コミュニティです。ファンアート、ファンフィクション、楽曲、配信、ファンイベントなど、さまざまな形でWeatherPlanetの魅力を広げ、共有することを目的としています。
