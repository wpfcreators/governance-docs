# 利用者

WPFCは「利用者」という用語を以下のように定義します。

**利用者**とは、WPFCが設けたあらゆるリソース、コンテンツ、サービス、またはコミュニケーション空間を閲覧・参加・使用する個人や団体を指します。これには、定義文書の閲覧者、SNSでの発信を読む者、ツールやサービスを使用する者、イベントに参加する者などが含まれます。

WPFCの定義において「利用者」は、単に受け手という受動的な立場にとどまらず、以下のような形でWPFCの活動と関わりを持つ存在と見なされます：

- **参加の形を問わない包括的定義**：WPFCが発信する情報やコンテンツに何らかの方法でアクセスし、利用するすべての人々を含みます。これには匿名の閲覧者、受動的な参加者、明示的な関与を行わない者も含まれます。
- **コンテンツに対する受容と反応の主体**：利用者は、WPFCの発信や作品に対して何らかの反応や受容の姿勢を持つ主体として位置づけられ、その解釈や反応がコミュニケーションの一部を構成するものと考えられます。
- **利用の態度と責任**：WPFCは、利用者に対して一定のリテラシーと理解を期待しています。特に、創作物の意図や趣旨を誤解・誤用せず、他者に配慮した利用を行う姿勢が望まれます。
- **双方向性の前提**：利用者は単に消費者ではなく、意見・感想・改善提案などを通じてWPFCの活動に影響を与えうる存在でもあると認識しています。

このようにWPFCは、「利用者」という語を単なる受け手としてだけでなく、WPFCの活動を成立させる重要な構成要素として定義します。利用者の存在と反応があることで、WPFCの表現活動は初めて公共性と意味を持ちます。

{: .highlight }
> この定義はWPFCが活動の前提として定めたものであり、WeatherPlanet公式の意見や方針を示すものではありません。
